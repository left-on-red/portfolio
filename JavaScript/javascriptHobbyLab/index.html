Node.js 101

Getting Started

Introduction

JavaScript is a vast language with many capabilities as you may know. And you may also know that you're mainly meant to write JavaScript for the browser. But JS shall no longer be limited to such restrictions. Imagine JavaScript with all of it's capabilities, but allowed to run outside of the web and as it's own standalone script like you would find with python for example. That is essentially what Node.js is. In addition to this, core functionality is given to the runtime to allow for things that JS wouldn't normally be capable of such as File IO, bindings to native cpp code, and much more.

Instructions

  2. Mozilla Developer Network is also a really good resource for just JavaScript in general as Node.js is just more feature rich JavaScript. But either way, it's very important to be well versed in JavaScript if you wish to succeed with Node.js
